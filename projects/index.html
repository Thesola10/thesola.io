What I've been working on

I'm passionate about software, especially free and open-source. Below are the things I built in my free time.

nixie  

Nixie installs a ./nix script in your repository, so you can ship Nix with your project, and let everyone work on the same page!

It's got a few tricks up its sleeve, like project templates, command shortcuts, and it's fully rootless!


Check it out, and if you're already using it, tell me about it so I can make a project gallery!

zsh comma assistant 

Comma is a tool that reads the Nixpkgs index to figure out where to find the command you want to run. Well, I put it directly into zsh so you'll never get a "command not found" error ever again.
